About ASUS Cloud

The ASUS Cloud is responsible for cloud service development at ASUS. With US$300 million capital, it is committed to the marketing, operations, and R&D of cloud Platform. A comprehensive range of services has been developed for individuals, families, businesses, education and healthcare services. ASUS Cloud also has open cloud resources to seek cooperation with partners in different disciplines, speed up industry integration and innovation, and help enrich the apps and services provided by the Taiwan cl0ud ecosystem.

Global Development

• 40 million users worldwide Consumer and Commercial) • Using multiple data centers around the world (Taipei/Shanghai/Los Angeles/Luxembourg) • Petabytes of files are stored in ASUS Cloud • Multiple service models (Public Cloud/Hybrid Cloud/Private Cloud) • ISO 27001 Information Security Management System Certification

Core Competence / ASUS Cloud Platform

ASUS Cloud Platform is a software-defined, data-driven cloud platform built using in-house technology. It has large amounts of data storage and powerful computing capabilities, to enable the development of cloud, mobile, big data and IoT applications. Products / ASUS WebStorage

ASUS WebStorage provides cross-device sync, backup, file sharing and keyword search features, bringing the convenience of cloud-based storage to consumers.
• Provides users with 5GB of free storage, with flexible options for additional storage.
• Supports seven operating systems and four browsers for comprehensive multi-device file access.
• ASUS WebStorage also provides convenient free services including a cloud file manager, a note-taking app, a wizard for sending large files, and a cloud printing service.

Products / ASUS FileCloud

ASUS FileCloud enables businesses to build an industry-leading enterprise collaboration environment.
• With a tailor-made private cloud, and simplified cloud platform configuration, FileCloud has an easy platform installation and setup that helps IT personnel deploy an enterprise-grade document cloud for users. • Integrates cloud storage, collaboration, backup synchronization, cloud search and cloud printing.
• Powerful management and easy-to-use interface help enterprises allocate cloud storage resources, manage data and integrate Microsoft AD/LDAP to enhance group permission controls.

Products / ASUS Classroom

ASUS Classroom gives teachers a platform to design, aggregate, and deliver teaching resources, track students’ responses, and enable interactive teaching at school or via home study.
• Create a class: Teachers can upload lesson materials and design teaching activities.
• Interaction in the classroom: Teachers can assess students using true-or-false questions, single choice, multiple choice, fill-in-the-blank, matching, assessment, discussion, homework and whiteboard materials, and immediately view the results in table and chart form.
• Pre-study / Homework: Teachers can flag activities as pre-class or after-class.
• The Learning Record Store keeps detailed records to help teachers track students’ performance via big-data analysis.
